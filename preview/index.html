DappKitty Preview

This is a demo of DappKitty using default settings. To activate the environment tools and logging panel, reload this page with one of the following query parameters:

  • Reload with ?kittyenv=dev
  • Reload with ?kittyenv=local

Quick Start Code

import dappKitty from 'dappkitty';
dappKitty('debug', {
  productionUrl: 'https://your-production-site.com'
});

Try It:
Open your browser's developer tools (usually F12 or right-click â†’ Inspect), then type: 

console.info('Test message from console.info');

Features in this Preview

  • Environment override via ?kittyenv=
  • Window/API/theme config simulation
  • Floating log panel with live console + fetch output
  • Error tracking (uncaught and unhandled)
  • Light/dark mode themes
